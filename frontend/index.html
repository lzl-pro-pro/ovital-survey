欢迎使用奥维野外调查插件

操作步骤：

  1. 新建项目，导入CAD文件（DXF/DWG），自动解析标注
  2. 审核标注，确认生成调查点（CAD坐标自动转经纬度）
  3. 点击「导出到奥维」→ 奥维"系统 → 导入对象" → 点标记显示在地图上
  4. 逐一调查：点击「一键定位」获取GPS → 填写信息 → 拍照
  5. 导出Excel：统计汇总表 + 单张原始记录表

支持DXF和DWG格式；导出到奥维为KMZ格式含照片
